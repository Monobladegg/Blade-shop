
### Stack

- `React` + `Typescript` + `Redux Toolkit`
- `React-router-dom` + `React-icons` + `Saas (modules)` + `Axios`
- `Yarn` + `Aliases` + `JSON Server` + `Husky`
- `Vite`
- `GitHub Pages`

### How to use

1. Download project
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:5173/blade-shop`
